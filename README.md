# Setup MacOS

## Contains

- .vimrc
- .zshrc
  - Install zsh
  - Install powerline
  - Install powerline fonts
  - Install rbenv
  - Install zsh-syntax-highlighting
- .tmux.conf
- authorized_keys
  - Macbook Pro
- scripts
  - forgit (fzf support for git)
- gitconfig
- config: stores .ssh file (~/.ssh/config)
